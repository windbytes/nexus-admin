# 预览功能快速入门 🚀

## 5分钟上手指南

### 第一步：点击预览按钮

在端点配置页面，点击底部的 **「预览」** 按钮（带有眼睛图标 👁️）

### 第二步：查看动态表单

预览弹窗会自动打开，展示根据配置生成的表单：

- **表单视图**：可以填写和测试的真实表单
- **JSON 视图**：完整的配置数据

### 第三步：填写并验证

1. 在表单中填写测试数据
2. 点击 **「验证并查看数据」** 按钮
3. 系统会验证表单并展示填写的数据

### 第四步：切换模式（如果有多个模式）

点击顶部的 Tab 标签（IN、OUT、IN_OUT、OUT_IN）查看不同模式下的表单

## 使用场景

### 🆕 新增端点时预览
```
1. 点击「新增端点类型」
2. 填写类型名称、编码等基础信息
3. 添加 Schema 字段
4. 点击「预览」查看效果
5. 根据预览结果调整配置
6. 满意后点击「保存」
```

### ✏️ 编辑端点时预览
```
1. 选择一个端点类型
2. 点击「编辑」
3. 修改配置
4. 点击「预览」查看修改后的效果
```

### 👀 查看已有配置
```
1. 选择一个端点类型
2. 直接点击「预览」
3. 查看生成的表单
```

## 表单组件支持

预览功能支持以下组件类型：

- ✅ **Input** - 输入框
- ✅ **InputPassword** - 密码输入框  
- ✅ **InputNumber** - 数字输入框
- ✅ **TextArea** - 文本域
- ✅ **JSON** - JSON 编辑器
- ✅ **Select** - 下拉选择
- ✅ **Radio** - 单选框
- ✅ **Checkbox** - 复选框
- ✅ **Switch** - 开关
- ✅ **DatePicker** - 日期选择器

## 配置示例

### 示例 1：简单输入框
```json
{
  "field": "name",
  "label": "名称",
  "component": "Input",
  "properties": {
    "placeholder": "请输入名称",
    "maxLength": 50
  },
  "defaultValue": "",
  "mode": "IN"
}
```

### 示例 2：下拉选择
```json
{
  "field": "type",
  "label": "类型",
  "component": "Select",
  "properties": {
    "placeholder": "请选择类型",
    "options": [
      { "label": "类型A", "value": "A" },
      { "label": "类型B", "value": "B" }
    ]
  },
  "mode": "IN"
}
```

### 示例 3：条件显示
```json
{
  "field": "password",
  "label": "密码",
  "component": "InputPassword",
  "showCondition": "formValues.needAuth === true",
  "mode": "IN"
}
```
> 只有当 `needAuth` 字段为 `true` 时，密码字段才显示

## 常用操作

### 🔄 重置表单
点击底部的 **「重置表单」** 按钮，清空所有填写的数据

### 💾 验证数据
点击 **「验证并查看数据」** 按钮，验证表单并查看填写的数据

### 🔀 切换视图
点击底部左侧的 **「表单视图」** 或 **「JSON视图」** 按钮切换

### 📋 切换模式
点击顶部的 Tab 标签在不同模式间切换

---

## 小贴士 💡

1. **编辑时预览**：编辑模式下预览使用当前编辑的数据（未保存）
2. **至少一个字段**：预览前需要至少配置一个 Schema 字段
3. **验证提示**：如果基础信息未填写完整，会提示先完善
4. **模式过滤**：每个模式只显示对应 mode 的字段
5. **排序显示**：字段按 sortOrder 升序排列
